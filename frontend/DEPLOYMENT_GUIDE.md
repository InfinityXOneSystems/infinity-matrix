
# Deployment Guide: Infinity X
**Target Environment:** Production (Cloud Agnostic / AWS / Railway)

---

## 1. Prerequisites
*   **Node.js:** v20.x or higher
*   **Docker:** v24.x
*   **npm/yarn:** Latest
*   **Git:** Latest
*   **Cloud Account:** AWS, Vercel, or Railway

---

## 2. Environment Variables
Create a `.env` file in the root directory. **DO NOT COMMIT THIS FILE.**
